# Masazai.net 

Live site: [https://www.masazai.net](https://www.masazai.net)

## About the Project

This project isn't just a basic React website — it's the result of a two-month mentorship program I led for an entry-level developer (Albertas). The project became a hands-on training ground where I taught the full development lifecycle using a real-world client scenario.

## Mentorship Focus

I personally guided Albertas through the fundamentals and real-world applications of:

-  **Git & GitHub** – version control, branching, pull requests
-  **React** – setting up and building components from scratch
-  **CSS & HTML** – responsive design and styling best practices
-  **Jira** – managing tasks, sprints, and versioned deliverables
-  **JavaScript** – functional logic, event handling, component state

We built everything collaboratively from scratch, with a strong focus on code readability, best practices, and hands-on problem-solving.

##  Personal Motivation

This project was close to my heart. I mentored an entry-level developer (Albertas) completely pro bono — while building a website for my mum’s business.

It was a “three birds with one stone” kind of mission:

- Learn how to teach and mentor effectively
- Create a website for my mum to help her business
- Help a beginner developer grow toward Juniority
